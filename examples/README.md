# Examples

Reference implementations from the dAIsy Chain factory pilot run (IBuySpy migration project).

## workflows/ibuyspy/
Deployment workflow examples for all 5 IBuySpy pilot apps across S3-S5:
- App Service deployments (Classifieds)
- IIS VM rehost deployments (Jobs)  
- Azure Container Apps deployments (TimeTracker, IBuySpy, PetShop)
- S5 infrastructure provisioning (Bicep-based)
- Post-deploy verification and smoke testing

When adopting this template for your own migration project, use these as starting points and adapt to your org, app names, and Azure topology.
